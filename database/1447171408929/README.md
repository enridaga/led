<p>My first memory is seeing the hall under construction; the next is listening to the opening broadcast concert. I have been fortunate to attend a number of concerts over the years but my earliest memories were the BBC Festival of Light Music Concerts, and a concert given by Mantovani. The most unusal memory was seeing EROS erected in the R.F.H.</p>
