<p class="MsoNormal" style="mso-pagination: none; mso-layout-grid-align: none; text-autospace: none;"><span lang="EN-US" style="mso-bidi-font-family: Arial; mso-ansi-language: EN-US;">I went last night to the opera,</span><span lang="EN-US" style="mso-bidi-font-family: Arial; mso-ansi-language: EN-US;"> and witnessed three things which were new to me.
