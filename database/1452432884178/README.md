<p>In the evening I went to the semi-serious opera that is Puccini's&nbsp;<em>Tosca&nbsp;</em>which, it must be admitted, contains some very expertly done things. The singers were not bad, the settings mundane, and the orchestra good. I looked for Lady Cunard but could not find her. Bored, I left after the second act.</p>
